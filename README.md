# TinyThread ![Build](https://travis-ci.org/cuhsat/tinythread.svg)
TinyThread `0.2.1`

(Ab)using TinyURL for hidden threads.

## Usage
```
tinythread.py [option|thread] [message...]
```

## Examples
Display the thread
```
$ tinythread.py foobar
```

Post a message
```
$ tinythread.py foobar "Hello World"
```
